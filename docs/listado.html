Añadir personaje

Como podemos ver ésta pantalla consta de varios botones para acceder a diferentes apartados, si pulsamos en el icono buscar situado arriba se abrirá la pantalla buscar, si clickamos en un personaje se abrirá la ventana modificar y pulsamos sobre el boton flotante se abrirá la ventana formulario para añadir un nuevo personaje. Ésta ventana consta de swipe (deslizar elementos hacia derecha o izquierda) para poder eliminar directamente el personaje seleccionado.
